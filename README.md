# 🛒 E-commerce Data Analysis System

## 📌 Project Overview
This project focuses on analyzing e-commerce data using SQL and visualizing insights using Power BI. It helps in understanding customer behavior, sales trends, and product performance.

---

## 🎯 Objectives
- Design and implement a relational database
- Perform data analysis using SQL queries
- Create interactive dashboards using Power BI
- Extract meaningful business insights

---

## 🛠️ Tools & Technologies
- MariaDB / MySQL
- Power BI Desktop
- SQL
- ODBC Connector
- CSV Dataset

---

## 🗂️ Database Structure
The project includes the following tables:
- Customers
- Products
- Orders
- Order_Details

Relationships:
- Customers → Orders
- Orders → Order_Details
- Products → Order_Details

---

## 📊 Features
- Sales Analysis Dashboard
- Category-wise Sales Visualization
- Price vs Quantity Scatter Analysis
- City-wise Sales Map
- Revenue Trend (Waterfall Chart)
- Funnel Analysis
